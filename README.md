# Graphical Password Authentication System

A secure graphical password authentication system built using Python Flask, allowing users to log in using images instead of traditional text passwords.

## Features:

- User registration with graphical password
- Login using image selection
- Flask web framework
- Simple and intuitive UI
- SQLite database for user storage

 ## Technologies Used:

- Python
- Flask
- HTML/CSS
- SQLite
- JSON

## Setup & Installation:

### 1️. Clone the Repository

git clone https://github.com/aashikohad/graphical-password-authentication-system.git

### 3. Navigate to Project Directory

cd graphical-password-authentication-system

### 3. Install Dependencies

pip install flask

### 4. Run the App
python app.py

### 5️. Visit in Browser

http://127.0.0.1:5000/
